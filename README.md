# Analysis of Statistical Functions Using Differential Methods

This repository contains a small student project exploring how **differential methods** can be used to analyze and better understand **statistical functions**.

## Project Goals

- Study how small changes in input data affect common statistical measures (mean, variance, standard deviation).
- Use **derivatives** and **differentials** to describe the sensitivity of these measures.
- Visualize the relationship between analytical results and numerical experiments.

## Methods

- Start from analytical formulas for basic statistical functions.
- Compute **partial derivatives** with respect to individual data points.
- Compare theoretical differentials with:
  - finite-difference approximations,
  - small random perturbations of the dataset.

## Technologies

- **Python** with:
  - NumPy for numerical computations,
  - Matplotlib for basic plots,
  - Jupyter Notebooks for interactive exploration.

## Status

This is an educational project, not a production-ready library.  
The main goal is to build intuition about how statistical quantities react to small changes in data.
